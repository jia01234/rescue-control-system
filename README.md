# 安全官搜救小隊人員工作管制系統

這是一套專為**前進指揮所安全官**設計的「離線優先、行動優化」人員工作管制與安全追蹤系統。主要功能包含管制危險區域人員出入、發布撤離訊號（含音效模擬）、記錄現場安全日誌，以及提供重要報表填寫之智慧提醒。

---

## 系統特色

1. **行動裝置優化 (Mobile-First)**
   - 採用直覺、寬大的觸控卡片與底部滑動選單（Bottom Sheet），便於戴手套或在晃動環境下以手機或平板單手操作。
2. **完全離線可用 (Offline-First)**
   - 採用純前端 HTML5 + CSS3 + Vanilla JS 技術，無須依賴伺服器或網路。
   - 所有資料（小隊名單、管制狀態、安全日誌）皆自動且即時儲存於瀏覽器之 `LocalStorage`，即使重新整理網頁、關閉瀏覽器甚至關機，資料均不會遺失。
3. **嗶聲撤退訊號模擬**
   - 內建基於網頁 Web Audio API 的音波合成器，可免下載音檔直接利用手機揚聲器播送：
     - **全面撤離 (Evacuate)**：三短音（每聲持續約1秒）。
     - **停止作業／保持安靜 (Cease Operations / Quiet)**：一長音（持續約3秒）。
     - **恢復作業 (Resume Operations)**：一長一短音。
4. **一鍵「熱區點名」**
   - 右下角懸浮按鈕可隨時點擊，即時過濾並顯示當前正在「熱區（危險工作區）工作」的人員清單、進去多久時間，並提供一鍵「全員撤離」快捷指令。
5. **智慧安全提醒**
   - **無線電定期回報 (Radio Check)**：每 20 分鐘計時器提醒，時間到時會發出警示音。
   - **報表填報提醒**：有人進入熱區時自動提示填寫「工作場地評估表」，有人返回時提示填寫「受困者解救表」，完成申報後可勾除提醒。
6. **日誌匯出**
   - 自動記錄人員進出、調動與警報日誌，安全官亦可打字輸入現場記事。日誌能一鍵匯出為標準 CSV 檔，利於任務結束後的存檔與檢討。

---

## 如何使用？

### 1. 啟動系統
- **離線啟動**：在電腦、平板或手機上，直接以任意瀏覽器（如 Chrome, Safari, Edge）開啟 `index.html` 即可。
- **無網環境**：可事先將整個目錄傳送至 iPad 或手機中，直接點選 `index.html` 執行。

### 2. 人員進出管制流程
- 預設分為**搜救 A 組**與**搜救 B 組**，可在頂部切換。
- 點選隊員卡片可開啟底部面板：
  - **移至熱區 (Hot Zone)**：隊員卡片會變為黃框紅字，並開始實時累計作業時間。超過 20 分鐘會開始閃爍紅色警報，提醒回報氣瓶或安全。
  - **移至已返回 (Returned)**：人員返回後，卡片會移至返回區，並鎖定該次熱區作業的總時間。
  - **移至待命 (Standby)**：人員休息完畢、重整後，可放回待命區以利下次部署。

### 3. 人員調動與新增
- 切換至「**人員名冊**」分頁。
- 可以直接在搜救 A 組、搜救 B 組、預備名單之間點擊「移至XX組」進行隊員調動。
- 點擊「**新增隊員**」可將新報到的搜救人員或專家直接編入名冊。

## 備份與清理

> [!WARNING]
> 因為資料存放在瀏覽器的 `LocalStorage`，若您**清除瀏覽器快取**或**使用私密瀏覽模式（無痕視窗）**，資料將會被抹除。
> 建議每次任務結束後，使用「日誌提醒」分頁中的「**匯出 CSV**」按鈕，將當次的安全日誌存檔。
